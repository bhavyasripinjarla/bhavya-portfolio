Smart Parking Detection using YOLOv5

This project detects parking slot occupancy using a combination of deep learning and computer vision techniques. A YOLOv5 model is used to detect vehicles in an image or video feed, and predefined parking slot coordinates are used to determine whether each slot is occupied or available. The system outputs a visual representation along with a CSV report of parking status.

The solution improves reliability compared to traditional contour-based methods by leveraging object detection and spatial overlap logic.

Features
  • Detects cars, bikes, buses, trucks
  • Determines occupied vs free slots
  • Works on image or video
  • Exports CSV report
  • Visual overlay with real-time bounding boxes

AirGuard: IoT Smart Wearable

An all-in-one smart wearable solution combining environmental awareness and health monitoring.

AirGuard is an IoT-based smart wearable device designed to monitor both environmental and health parameters in real time. It integrates air quality sensing with health monitoring features into a compact smartwatch-like prototype, enabling users to stay aware of their surroundings and physical condition.

Key Features
  • Real-Time Air Quality Index (AQI) Monitoring
  • Heart Rate & Oxygen Level (SpO₂) Tracking
  • Time, Battery, WiFi & Bluetooth Status
  • Music Control & Activity Tracking
  • Smartwatch UI with Real-Time Updates
Architecture

The system uses ESP32-S3 as the central controller. Environmental data is collected using the BME680 sensor, and health data is captured using the MAX30102 sensor. The ESP32 processes the data and displays it on the smartwatch interface in real time.

Hardware Prototype
  • Compact smartwatch prototype with 3D-printed enclosure
  • Magnetic charging mechanism
User Interface & Results

Real-time AQI, heart rate, and SpO₂ visualization with clean circular UI design.

SmartSecureNet

SmartSecureNet is an AI-powered system designed to analyze public Wi-Fi security and detect malicious networks. It combines mobile-based Wi-Fi analysis with desktop-level network monitoring to provide real-time threat detection.

Key Features
  • Wi-Fi Network Scanning & Encryption type detection
  • AI-Based Security Risk Analysis & Evil Twin Detection
  • Real-Time Packet Monitoring (Captures network packets, detects ARP/DNS attacks)
  • Security Score System & Interactive Dashboard
Architecture

The system consists of a mobile app for Wi-Fi scanning and a desktop system for real-time traffic monitoring. Data is processed using machine learning (Random Forest on CICIDS2017) to detect threats.
